# Textarea Autosize

Autosizes textarea to size of its contents.

## Installation

```
$ npm install @github/textarea-autosize
```

## Usage

```js
import autosize from '@github/textarea-autosize'
autosize(document.querySelector('textarea.foo'))
```

## Browser support

- Chrome
- Firefox
- Safari
- Internet Explorer 11
- Microsoft Edge

## Development

```
npm install
npm test
```
